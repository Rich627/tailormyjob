## Submission Checklist

For each list:

- [ ] Fork the repository
- [ ] Read their CONTRIBUTING.md guidelines
- [ ] Add entry in alphabetical order (if required)
- [ ] Follow their formatting style
- [ ] Create pull request with clear title
- [ ] Wait for review and address feedback
